# FaceRecognition
Face Recognition Classification using ML methods: Bayes Classifier, KNN, Kernel SVM, Boosted SVM, PCA and MDA

## Tasks
* Task 1: For dataset DATA(200 classes), POSE(68 classes) and ILLUMINATION(68 classes), identify the person(class) from each dataset.
	* * Use KNN and Bayes classifier.
* Task 2: For dataset DATA(2 classes, 400 samples) classify neutral vs expression.
	* * Kernel SVM, Boosted SVM, KNN, Bayes Classifier
